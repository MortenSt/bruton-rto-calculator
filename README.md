# Bruton × 2020 Bulkers — RTO Dilution Calculator

Interaktiv kalkulator for å analysere dilution-scenarioer ved en potensiel reverse takeover (RTO) mellom Bruton Limited og 2020 Bulkers Ltd.

**English**: Interactive calculator to analyze dilution scenarios in a potential reverse takeover (RTO) between Bruton Limited and 2020 Bulkers Ltd.

---

## About / Om prosjektet

I september 2025 kunngjorde Tor Olav Trøim planen om å bygge Bruton til et Frontline-lignende tankrederi (VLCC-seiler). Bruton er notert på Euronext Growth Oslo med mål om NYSE-notering i 2026.

**Hypotesen**: Magni Partners kan bruke 2020 Bulkers' 7-års hovedlistekreditering til å akselerere NYSE-listingen via en RTO midt 2026.

**Fordelene**:
- Institusjonell investor tilgang (pensjonsfond, forsikring, ASK-konto)
- Billigere kapitalkostnader på hovedliste/NYSE vs Growth  
- Arvet utbyttekreditering fra 2020B (78 måneder kontinuerlig profittabilitet)
- Forkortelse av NYSE-listeringstidslinja med 12-18 måneder

**Denne kalkulatoren**: Test ulike shell-verdier og se dilution-matematikken.
